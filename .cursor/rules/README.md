# Cursor Rules

このディレクトリには、Cursor AIがプロジェクトを理解し、適切なコードを生成するためのルールが含まれています。

## ルールファイル

- **[poker-game-rules.md](poker-game-rules.md)**: ポーカーゲームの実装ルールとベッティングラウンドの完了条件
- **[code-style.md](code-style.md)**: コードスタイルとベストプラクティス
- **[architecture.md](architecture.md)**: アーキテクチャの理解ガイド
- **[project-context.md](project-context.md)**: プロジェクトの概要とコンテキスト

## 使用方法

これらのルールは、Cursor AIがコードを生成・修正する際に自動的に参照されます。新しいルールを追加する場合は、このディレクトリにMarkdownファイルとして追加してください。

## 重要なルール

### ベッティングラウンドの完了条件
- プリフロップ: BBがアクションを取った後、アクションがBBに戻ってきた時点で完了
- フロップ以降: アクションが開始位置に戻ってきた時点で完了

### 最小レイズ額の計算
- `min_raise_amount = max(現在のベット額 + 直前のレイズ幅, 現在のベット額 × 2)`

詳細は各ルールファイルを参照してください。
